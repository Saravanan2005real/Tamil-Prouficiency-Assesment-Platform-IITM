வழிமுறைகள் / Instructions

  • தமிழ்: தமிழ் மற்றும் ஆங்கிலம் இரண்டிலும் பகுதியை கவனமாக படிக்கவும்.
    English: Read the passage carefully in both Tamil and English.
  • தமிழ்: அனைத்து கேள்விகளுக்கும் தமிழில் மட்டும் பதிலளிக்கவும்.
    English: Answer all questions in Tamil only.
  • தமிழ்: தமிழ் விசைப்பலகையை பயன்படுத்தவும் (பரிந்துரைக்கப்படுகிறது) - தட்டச்சு செய்ய விசைப்பலகை பொத்தானை கிளிக் செய்யவும்.
    English: Use Tamil Keyboard (recommended) - click the keyboard button to type.
  • தமிழ்: அல்லது ஆங்கிலத்திலிருந்து தமிழுக்கு தட்டச்சு செய்ய Google Input Tools Chrome Extension-ஐ நிறுவவும்.
    English: Or install Google Input Tools Chrome Extension for English-to-Tamil typing.
  • தமிழ்: ஒவ்வொரு பதிலும் சரியானதா என்பதைப் பார்க்க "மதிப்பீடு" என்பதை கிளிக் செய்யவும்.
    English: Click "Evaluate" for each answer to see if it is correct.
  • தமிழ்: மொத்தம் 3 பகுதிகள் (15 கேள்விகள்).
    English: Total 3 passages (15 questions).
