05. Internal Architecture (1/2)

Neural Evaluation Physics

Speaking Engine

Qwen 2.5 • Audio Vector Logic

Relevance Gate (Qwen)

Executes a locked semantic prompt forcing Qwen to extract the "Answer Topic" vs "Question Topic" vectors. Fails the entire response pipeline immediately if the contextual topic match falls below 30%.

Confidence Mapping

Extracts Root Mean Square (RMS) energy across precise 50ms audio frames. Calculates standard deviation. Instantly penalizes the score mathematically if volume drops or hesitations exceed the >40% volume ratio threshold, operating independently of the transcribed text.

Fluency & Pace Target

Calculates timestamped Words Per Minute (WPM) targeting specific optimal Tamil phonetic speeds (110-150 WPM). Subtracts targeted score points for any syllable separation gaps exceeding 1.5 seconds calculated computationally.

Coherence Architecture

A specific algorithm checks Tamil logical connector thresholds (e.g., "மேலும்", "எனவே") and explicitly maps whether sentences contain grammatical conclusion points, verifying structural flow independent of AI topic relevance.

Writing Engine

90MB Lexicon array • Morphological Slicer

Immediate Hash Lookup

Cleans trailing Unicode spaces internally and matches every entered token parallel-wise against a customized 90MB compiled valid Tamil lexical array executing in under < 0.05ms string speeds.

Morphological Spell Slicing

Splits compounded user strings into pure Root + Suffix computationally (e.g. பள்ளிக் + கு). Validates both the dictionary base root and the exact Tamil grammatical conjunction dependencies independently before reconstructing spelling scores.

Semantic Category Filter

Analyzes concept structures locally (blocking attempts to apply "physical touch" verbs strictly to abstract nouns like "knowledge") avoiding irrelevant LLM costs before verifying complex tense grammar using the Ollama parameter network.

Dynamic Density Weight

Scores vocabulary algorithmically based on active density. Repeated base terms mathematically yield a 0.0 multiplier, whereas unique advanced descriptors generate positive percentage spikes.

05. Internal Architecture (2/2)

Neural Evaluation Physics

Listening Engine

Multi-Lingual MiniLM Context Matching

Localized Edge Pipeline

Converts specific audio wavelengths directly to text utilizing Whisper models. The logic instantly strips punctuation constraints and standardizes all casing down to pure conceptual base strings.

Logical Trap Enforcement

Specifically searches transcribed strings against hardcoded negation-entity arrays (like "இல்லை"). Blocks positive semantic matching logic instantly if the core intent is identified as mathematically reversed versus the required answer key.

MiniLM Latent Vector Matching

Encodes both the target master answer and the user's transcribed string into heavily dimensional embeddings using exactly MiniLM-L12-v2 multi-lingual nodes. Legitimizes true structural meaning overlap via Cosine distance vectors requiring > 85% overlap parameters to output success states.

Reading Engine

Abstract Structural Integrity

Specific Fact-Vector Regex

Javascript engine extracts specific hard facts separate from generalized text blocks. Dates, Locational identifiers, and core nouns are independently locked via regex matrices, bypassing standard LLM hallucination error rates entirely.

Qwen Conceptual Parity Network

Validates deep comprehension over localized copy/pasting behavior. If a user rewrites an entire paragraph leveraging altogether different Tamil synonyms, Qwen's latent space measures conceptual parity dynamically; it links underlying abstracted semantics logically back to the paragraph's core reference anchors.

Internal Contradiction Sweeps

Models specifically review the generated linguistic constraints identifying self-contradictory logic loops. A user asserting "the plan succeeded", but later declaring "the same plan failed completely", initiates a major mathematical drop inside the final grading vector calculations.
