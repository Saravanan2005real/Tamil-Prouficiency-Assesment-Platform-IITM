Technical Specifications

Component Sub-system / Algorithm Primary Function
Presentation React + TailwindCSS + HTML5 Canvas Real-time user rendering and multimodal capture.
Logic Controller Node.js / Python (FastAPI) Session state management and score aggregation.
NLP Core Tamil BERT-Base (Pre-trained) Semantic analysis, grammar check, and syntax scoring.
Audio Engine Wav2Vec 2.0 / DTW Align Tamil phoneme mapping and pronunciation grading.
OCR Module Attention-based LSTM / ResNet Handwritten Tamil script recognition (Writing skill).
Storage MongoDB / PostgreSQL Secure storage of officer certification records.
