# Google Input Tools Fix Guide

## If Google Input Tools is not working:

### Method 1: Check Browser Console
1. Open browser Developer Tools (F12)
2. Go to Console tab
3. Look for messages starting with "✓" or errors
4. Check if you see "Google Input Tools script loaded"

### Method 2: Manual Activation
1. Click "Google Input (English to Tamil)" button
2. Wait for button to show "Active"
3. Click in the textarea
4. Type in English (e.g., "namaskaram")
5. Press Space or Enter to convert

### Method 3: Alternative - Use Tamil Keyboard
If Google Input Tools doesn't work, use the Tamil Keyboard:
1. Click "Tamil Keyboard" button
2. Click letters to type
3. Hover over letters to see variations
4. Click variation buttons to insert

### Method 4: Browser Extension
Install Google Input Tools Chrome Extension:
1. Go to Chrome Web Store
2. Search "Google Input Tools"
3. Install the extension
4. Enable Tamil transliteration
5. Use Ctrl+Shift to toggle

### Troubleshooting:
- **Check internet connection** - Google Input Tools requires internet
- **Check browser console** for errors
- **Try refreshing the page**
- **Try a different browser**
- **Check if ad blockers are blocking the script**
